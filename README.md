# CMSC21: Hiraeth

## Compiling
### Windows (on Shell)
```bash
g++ -std=c++17 -o main.exe main.cpp
```

### Windows (on Visual Studio Code)
Click build on the top right corner.

### Linux (on Shell)
```bash
g++ -std=c++17 -o main main.cpp
```

## Running
### Windows (on Shell)
```bash
./main.exe
```

### Windows (on Visual Studio Code)
Click build and run on the top right corner.

### Linux (on Shell)
```bash
./main
```
